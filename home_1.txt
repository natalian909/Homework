1) Посмотреть где я - pwd
2) Создать папку - mkdir jerry
3) Зайти в папку - cd jerry
4) Создать 3 папки - mkdir tom1 tom2 tom3
5) Зайти в любую папку - cd tom1
6) Создать 5 файлов (3 txt, 2 json) - touch 1.txt 2.txt 3.txt 4.json 5.json
7) Создать 3 папки - mkdir bom1 bom2 bom3
8) Вывести список содержимого папки - ls -la
9) + Открыть любой txt файл - vim 1.txt
10) + написать туда что-нибудь, любой текст. - "i" ввести текст
11) + сохранить и выйти. - esc :wq enter
12) Выйти из папки на уровень выше cd ..
    на 2 уровня cd ../..
    на 3 cd ../../.. и т.д.
13) переместить любые 2 файла, которые вы создали, в любую другую папку. - 
    mv /d/jerry/tom1/1.txt 2.txt /d/jerry/bom1 (если выходишь из директории)
    mv ./tom1 1.txt 2.txt ../bom2 (вариант не срабатывает!!!)
    mv 1.txt 2.txt bom1 (если в этой же директории)
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - 
    cp /d/jerry/tom1/1.txt 2.txt /d/jerry/bom1 (если выходишь из директории)
    cp ./tom1 1.txt 2.txt ../bom2 (вариант не срабатывает!!!)

15) Найти файл по имени - find -name 1.txt
16) просмотреть содержимое в реальном времени - tail -f 1.txt 
17) вывести несколько первых строк из текстового файла - head -n(нужное кол-во) 1.txt
18) вывести несколько последних строк из текстового файла - tail -n(нужное кол-во) 1.txt
